## Overview

This repository is a comprehensive portfolio of Natural Language Processing projects, spanning classical NLP techniques through modern transformer-based architectures. Each project demonstrates practical applications of language understanding and generation, from basic text processing to cutting-edge large language model fine-tuning.

The goal is to build deep expertise across the full NLP spectrum -- tokenization, embeddings, sequence models, attention mechanisms, and LLM workflows.

## Getting Started

### Prerequisites

```bash
pip install torch transformers spacy nltk gensim scikit-learn jupyter
python -m spacy download en_core_web_sm
```

### Running Notebooks

```bash
jupyter notebook
```

Navigate to the topic folder of interest and open the corresponding `.ipynb` file.
